# Code of Conduct

We are committed to providing a friendly, safe, and welcoming environment for all.

Our Pledge
- Be respectful and inclusive.
- Offer constructive feedback.
- Accept responsibility and apologize when appropriate.

Unacceptable Behavior
- Harassment, discrimination, or hateful conduct.
- Personal attacks or derogatory comments.
- Disruptive or aggressive behavior.

Enforcement
- Report incidents privately via GitHub Security Advisories or by contacting the maintainers.
- Maintainers may take any action they deem appropriate, including warnings or removal from the community.

Scope
- This Code applies within project spaces and when representing the project in public spaces.

This policy is adapted from common open-source community guidelines (e.g., the Contributor Covenant).
